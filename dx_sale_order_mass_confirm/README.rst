================================
Confirmar Pedidos Masivamente
================================

Permite la confirmación de pedidos de manera masiva.

Uso
===

1. Ir a Ventas > Cotizaciones
2. Seleccionar varios pedidos desde la vista de árbol.
3. Acción > Confirmar pedidos.
