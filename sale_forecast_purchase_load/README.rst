===========================
Sale Forecast Purchase Load
===========================

Based on `procurement_purchase_forecast <https://github.com/odoomrp/odoomrp-wip/tree/8.0/procurement_purchase_forecast>`_.

This module adds a new wizard on sale forecast to load purchase lines.

Depends on sale_forecast

Usage
=====

Sales -> Sale Forecast -> Load Purchases
